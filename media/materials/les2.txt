Функции в C# — это блоки кода, выполняющие определенную задачу.  
Основные особенности:  

1. Синтаксис:  
   [модификатор] [тип] [имя](параметры) { ... }  
   Пример:  
   public int Sum(int a, int b) { return a + b; }  

2. Модификаторы доступа:  
   - public    — доступен везде  
   - private   — только внутри класса  
   - protected — внутри класса и наследников  
   - internal  — в пределах сборки  

3. Статические функции:  
   - Принадлежат классу (не экземпляру).  
   - Пример:  
     static void Print() { Console.WriteLine("Hello"); }  

4. Возвращаемое значение:  
   - void — если функция ничего не возвращает.  
   - Любой тип (int, string, bool и т.д.).  

5. Параметры:  
   - Могут быть по значению, ссылке (ref/out) или через params.  


Пример вызова:  
int result = Sum(2, 3);
